# Meditation Website 

Questo progetto è una semplice applicazione web realizzata con **React** e **Vite**.
L’obiettivo è creare un sito di meditazione con un **timer**, una **home page** e una **pagina about**, utilizzando componenti React.

---

## Funzionalità principali

- Timer per la meditazione
- Layout semplice e centrato
- Navigazione tra le pagine (Home / About)
- Design responsive (adattabile a mobile)

---

## Tecnologie utilizzate

- React
- Vite
- JavaScript
- CSS
- React Router DOM

---

## Struttura del progetto

- `src/components` → componenti React (es. Timer)
- `src/pages` → pagine del sito (Home, About)
- `App.jsx` → componente principale
- `App.css` → stile dell’applicazione
